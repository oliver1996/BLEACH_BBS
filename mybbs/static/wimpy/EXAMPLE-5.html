IMPORTANT: The player will only work properly on a live web server!

The wimpy-iframe.html file is a special file that allows you to include wimpy player in another page (or web site) in an isolated manor.

Player options are configured with a ?query string tacked on to the end of the URL to the special wimpy.iframe.html file.

Note that when using an iframe, you do not need to include a reference to the wimpy.js engine.
